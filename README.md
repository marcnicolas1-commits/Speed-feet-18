# SpeedFeet Analyzer V2.3.0

Version complète de test terrain.

## Nouveautés principales
- Préparation enrichie : météo, configuration bateau, objectif et check-list.
- Événements horodatés et géolocalisés : vent, réglage, virement, empannage, envoi et affalage du spi.
- Synchronisation des événements avec une trace Velocitek VCC, décalage automatique et correction manuelle.
- Polaire de vitesse après navigation.
- Segmentation et comparaison des tests de réglages.
- Gestion des données : suppression d’une navigation, de toutes les navigations ou réinitialisation complète.
- Base locale et fonctionnement PWA hors ligne conservés.
