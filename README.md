# SpeedFeet Analyzer V2.2.0

Correctif terrain de la V2.

## Inclus
- Cartes de l’accueil cliquables.
- Dernière navigation ouverte directement depuis l’accueil.
- Liste complète des points à faire.
- Accès direct à Import / Export depuis l’accueil.
- Dictée vocale et saisie clavier dans les commentaires de marqueur.
- Bip de confirmation à l’ouverture et à l’enregistrement d’un marqueur.
- Ajout, remplacement et ouverture d’un VCC directement depuis une fiche du Carnet.
- Association d’un VCC à une navigation depuis Analyse.
- Icône iPhone / iPad personnalisée.

La synchronisation en ligne n’est pas encore incluse : elle sera ajoutée après validation de cette version locale.
