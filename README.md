# ⛵ SpeedFeet Analyzer

SpeedFeet Analyzer est une Progressive Web App (PWA) développée pour analyser les performances d'un voilier lors des entraînements et des régates.

L'application est pensée pour fonctionner entièrement hors ligne et permettre un enregistrement rapide des données directement à bord.

---

# Fonctionnalités

## Navigation

- Préparation d'une navigation
- Enregistrement GPS
- Carte interactive
- Calcul de la vitesse
- Calcul de la distance
- Temps de navigation
- VMG
- Marqueurs personnalisés

## Conditions météo

- Vent moyen
- Rafales
- Direction du vent
- État de la mer
- Capture d'écran météo
- Notes

## Réglages du bateau

Enregistrement rapide des réglages :

- Chariot de grand-voile
- Chariot de foc
- Rotation du mât
- Cunningham
- Bordure
- Écoute de grand-voile

## Historique

- Sauvegarde automatique des navigations
- Réouverture d'une navigation
- Consultation des statistiques
- Analyse des performances

## Paramètres

- Nom du bateau
- Configuration par défaut
- Import d'une polaire
- Import d'un VCC
